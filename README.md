UIImage+TextureAtlas
=================
UIImage category to convert XCode generated Texture Atlas files into UIImages. Useful for animations outside of SpriteKit.

More information coming soon.

Setting Up
=================
Via Cocoapods

	pod 'UIImage+TextureAtlas', '~> 0.1'
